# Product Guide

## Project Vision
A Gemini CLI extension that contains a modular suite of Gemini CLI extensions for the Software Development Lifecycle (SDLC). This project provides high-precision, single-responsibility AI agent skills that are designed to be atomic: each skill performs exactly one task.

## Target Audience
- **Individual Developers:** Who want to automate repetitive coding and documentation tasks.
- **Software Engineering Teams:** Looking to collaborate with AI and standardize their development processes.
- **Technical Leads and Architects:** Who need to enforce software quality and unify AI-assisted workflows across the SDLC.

## Core Goals
- **Increase Developer Productivity:** Automate repetitive and time-consuming tasks across the SDLC.
- **Enhance Software Quality:** Integrate AI-driven analysis, reviews, and audits directly into the development cycle.
- **Unify AI-assisted Workflows:** Provide a standardized set of AI tools that are consistent and predictable.
- **Atomic Precision:** Design each skill as a single-responsibility, atomic agent that performs one task exceptionally well.

## Core Features and Extensions
- **Coding and Refactoring Extension:** Intelligent code generation, complex refactoring, and style enforcement.
- **Testing and QA Extension:** Automated unit and integration test generation, execution, and debugging assistance.
- **Documentation and Release Extension:** Automatic generation of README files, API documentation, changelogs, and release notes.
- **Full SDLC Coverage:** Tools for analysis and design, requirements gathering and elaboration, and quality control.

## User Experience (UX)
- **Dedicated Namespace:** The suite is accessible through a dedicated command namespace (e.g., `gemini-suite <command>`) for clarity and separation of concerns.
- **Seamless Integration:** The extensions work together to provide a cohesive experience throughout the development lifecycle.
